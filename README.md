# Influencer Analytics Platform

A C++ based analytics system that analyzes influencer data and provides insights about followers, engagement rate, likes, and performance ranking.

## Project Overview

Influencer Analytics Platform helps analyze social media influencer data. It processes influencer information, calculates engagement metrics, and generates performance reports.

## Features

- Add influencer details
- Display influencer information
- Calculate engagement rate
- Compare influencer performance
- Rank influencers
- Generate analytics reports

## Technologies Used

- C++
- Object-Oriented Programming
- STL
- File Handling
- Git & GitHub
