# 📊 Influencer Analytics Platform

![Project Banner](screenshots/dashboard.png)

## 🚀 Overview

Influencer Analytics Platform is a data-driven application designed to analyze social media influencer performance using important engagement metrics.

The system processes influencer data such as followers, likes, comments, and category information to calculate performance insights, engagement rates, and identify top-performing influencers.

This project demonstrates the application of **Object-Oriented Programming, data processing, and analytical logic using C++**.

---

# 🎯 Objectives

* Analyze influencer performance using available data
* Calculate engagement rates
* Compare influencers based on performance metrics
* Generate meaningful analytical reports
* Build a foundation for future AI-based influencer prediction systems

# 🧮 Engagement Rate Formula

```
Engagement Rate =
((Likes + Comments) / Followers) × 100
```

Example:

```
Followers = 250000
Likes = 45000
Comments = 5000

Engagement Rate =
(50000 / 250000) × 100

= 20%
```
